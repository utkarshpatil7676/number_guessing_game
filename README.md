# number_guessing_game
### Project Description: Number Guessing Game with GUI (Tkinter)

The **Number Guessing Game** is a simple interactive desktop application developed using **Python** and the **Tkinter** library. The game randomly generates a number between 1 and 100, and the user must guess the correct number through a graphical user interface.

The application provides instant feedback on each guess by indicating whether the entered number is too high or too low. It also tracks the number of attempts taken by the user and displays a success message when the correct number is guessed. A reset option allows users to start a new game without restarting the application.

This project demonstrates the fundamentals of GUI development, event-driven programming, random number generation, input validation, and game logic implementation in Python. It is an excellent beginner-level project for understanding how to build user-friendly desktop applications.

### Technologies Used

* Python
* Tkinter (GUI Library)
* Random Module

### Key Features

* User-friendly graphical interface

* Real-time feedback on guesses
* Attempt counter
* Input validation
* New Game/Reset functionality
